# Code of Conduct

## Our Standard

All participants are expected to:

- be respectful and professional
- provide constructive technical feedback
- avoid harassment, discrimination, and abusive behavior
- respect confidentiality and proprietary project boundaries

## Unacceptable Behavior

- harassment, intimidation, or personal attacks
- hate speech or discriminatory language
- sharing private data, credentials, or confidential project details
- intentional disruption of collaboration processes

## Scope

This Code of Conduct applies to:

- repository discussions and issues
- pull requests and code reviews
- private channels used for project collaboration

## Reporting

Report violations to the project owner listed in [`LICENSE.md`](LICENSE.md).

Include:

- what happened
- where it happened
- relevant links/screenshots
- any immediate safety concerns

## Enforcement

The project owner may take any action deemed appropriate, including warning, temporary restriction, or permanent removal from project collaboration.

## Licensing Boundary

Participation in discussions or submissions does not grant any rights beyond the Proprietary Commercial License. See [`LICENSE.md`](LICENSE.md).
