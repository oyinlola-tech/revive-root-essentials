# Support

For licensing, enterprise usage, customization, and support terms, use the owner contact listed in [`LICENSE.md`](LICENSE.md).

When requesting support, include:

- environment (local/staging/production)
- exact error message and timestamp
- steps to reproduce
- relevant logs (redacted, no secrets)
- expected vs actual behavior

## Security Reports

For vulnerabilities, follow [`SECURITY.md`](SECURITY.md) and report privately.
